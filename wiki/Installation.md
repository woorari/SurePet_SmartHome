# Installation

> 🚧 The integration is in early development. These steps describe the intended
> install flow and will be finalized with the first release.

## Via HACS (recommended)

1. Install [HACS](https://hacs.xyz/) if you haven't already.
2. In HACS, open the menu (⋮) → **Custom repositories**.
3. Add this repository's URL with category **Integration**.
4. Search for **Surepet SmartHome** in HACS and install it.
5. Restart Home Assistant.
6. Continue with **[[Configuration]]**.

## Manual

1. Download the latest release archive.
2. Copy `custom_components/surepet_sh/` into your Home Assistant
   `config/custom_components/` directory.
3. Restart Home Assistant.
4. Continue with **[[Configuration]]**.
