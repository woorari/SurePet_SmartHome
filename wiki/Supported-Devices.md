# Supported Devices

| Device | Status |
|--------|--------|
| Hub | ✅ |
| Cat Flap Connect | ✅ |
| Pet Flap Connect | ✅ |
| Feeder Connect | ✅ |
| Feeder Lite | ✅ |
| Felaqua Connect (water station) | ✅ |
| Repeater | ✅ (signal / diagnostics) |
| Programmer | ✅ (signal / diagnostics) |

All devices are reached through your Sure Petcare **Hub**, which bridges them to
the cloud. The integration nests each device under its hub in Home Assistant.

> The integration talks to the Sure Petcare cloud — there is no local/offline
> mode. An internet connection and a working hub are required.
