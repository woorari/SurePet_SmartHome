# Sure Petcare for Home Assistant

Welcome to the wiki for the **Surepet SmartHome** integration — a custom Home
Assistant integration for Sure Petcare (SureHub) connected pet devices.

> 🚧 **Early development.** Not yet ready for production use.

## What it does

Brings your Sure Petcare hub, cat/pet flaps, feeders and water stations into
Home Assistant, with sensors for connectivity, battery, signal, pet location and
feeding/drinking activity — and automation-friendly features the built-in
integration doesn't offer.

> **Unofficial & unaffiliated.** Not affiliated with or endorsed by Sure
> Petcare. It uses the same cloud API as the official apps, which is
> undocumented and may change at any time.
